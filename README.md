# 🎬 Motivational Shorts Creator

AI-powered tool that generates **high-retention** motivational scripts for YouTube Shorts (55-60 seconds). Plus titles, descriptions, hashtags, and A/B testing.

---

## ✨ Features (All 100% Free)

### Script Generation
- 🧠 High-retention motivational scripts (55-60 seconds)
- 🔥 Powerful first hook (grabs attention in 0-3 seconds)
- 📝 Multiple script tones (Urgent, Inspirational, Emotional, Direct)
- ⏱️ Character counter to fit exactly 55-60 seconds
- 🎯 No duplicate scripts (AI generates unique content every time)

### YouTube Optimization
- 🧪 **A/B Testing** - Get 3 different title options
- 🏆 **High-CTR titles** - Tested for YouTube Shorts algorithm
- 📝 **Optimized descriptions** - First 2 lines hook the viewer
- 🏷️ **Hashtag strategy** - Mix of trending + niche tags
- 📊 **Best time to post** - Based on your target audience

### Content Management
- 📚 **Script History** - Save last 50 generated scripts (browser local storage)
- ⭐ **Favorites/Bookmarks** - Save your best scripts
- 🔍 **No duplicate titles** - Automatically checks previous titles
- 📤 **Export as TXT** - Download your script package

### Future Free Upgrades (Coming)
- 🔊 **Voiceover** (Edge TTS - No API key needed)
- 🖼️ **Stock footage integration** (Pixabay - Free)
- 🎬 **Video rendering** (Remotion - Free for individuals)
- 📤 **Direct YouTube upload** (YouTube API - Free)

## 📦 Setup Instructions

1. Clone the repository
2. Run `npm install`
3. Create `.env` file (template provided)
4. Get free API keys:
   - Groq: https://console.groq.com (30 seconds, no credit card)
   - Pixabay: https://pixabay.com/api/docs/ (2 minutes, free)
5. Run `npm start`
